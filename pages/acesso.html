Acesso ao Sistema

Neste módulo explicaremos como realizar o login de acesso ao EFESUS.


Para realizar o acesso ao sistema, siga os seguintes passos:

  1. Acesse a página inicial do sistema.

  2. Digite seu nome e-mail

  3. Insira sua senha.

  4. Clique em "Entrar".

Após realizar o acesso essa é a tela inicial que será apresentada para o usuário.
